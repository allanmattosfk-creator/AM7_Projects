# AM7 Projects

Pequena biblioteca de utilidades matemáticas em Python, usada como projeto de estudo.

## Funções disponíveis

- `add(a, b)` — soma dois números
- `subtract(a, b)` — subtrai dois números
- `multiply(a, b)` — multiplica dois números

## Rodando os testes

python -m unittest discover
